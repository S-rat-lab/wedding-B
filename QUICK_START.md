## Step 3: Deploy to GitHub Pages

### A. Create GitHub Repository (First Time Only)

1. Go to https://github.com/new
2. Repository name: `wedding-invitation`
3. Make it **Public**
4. Click "Create repository"

### B. Push to GitHub

```bash
# Add all files
git add index.html

# Commit
git commit -m "Set main invitation page"

# Add your GitHub repository (replace YOUR-USERNAME)
git remote add origin https://github.com/YOUR-USERNAME/wedding-invitation.git

# Push to GitHub
git push -u origin main
```

### C. Enable GitHub Pages

1. Go to your repository: `https://github.com/YOUR-USERNAME/wedding-invitation`
2. Click **Settings** tab
3. Click **Pages** in left sidebar
4. Under "Source": Select **main** branch
5. Click **Save**
6. Wait 1-2 minutes

### D. Your Link is Ready! ğŸ‰
